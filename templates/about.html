About ToxBase

What is ToxBase?

ToxBase is an open-source, community-driven toxicology database designed to collect, organize, and make accessible scientific data from toxicological studies. Our mission is to create a comprehensive repository of toxicology information that can be used by researchers, toxicologists, regulatory scientists, and the public.

By standardizing how toxicology data is collected and shared, ToxBase aims to facilitate more efficient analysis, comparison, and meta-analysis of toxicological findings across different studies and research groups.

Data Structure

ToxBase organizes toxicology data in a hierarchical structure:

  • Study - The top level that contains information about the research investigation, including the toxin studied, author, publication references, and description.
  • Animal Model - Details about the test subjects used in the study, including species, strain, sex, age, and weight.
  • Dose Group - Information about different exposure levels, including the dose value, unit, group size, and exposure duration.
  • Outcome - The observed effects or results from the study, such as cancer incidence, organ weight changes, mortality, or other endpoints.

This structure allows for precise data entry and efficient querying, enabling users to find specific information or analyze trends across multiple studies.

Community Contribution

ToxBase thrives on community participation. Anyone can contribute to the database by:

  • Adding new toxicology studies
  • Entering data from published literature
  • Improving existing entries with additional information
  • Suggesting enhancements to the database structure

To maintain data quality, we encourage contributors to:

  • Provide complete and accurate information
  • Include references to original sources
  • Follow the standardized data entry format
  • Document any assumptions or interpretations made when entering data

Open Source Project

ToxBase is fully open-source, built with Python using the Flask web framework and SQLAlchemy for database management. Our code is available for review, contribution, and forking.

Developers interested in contributing to the codebase can help by:

  • Adding new features
  • Improving the user interface
  • Enhancing database performance
  • Fixing bugs and addressing issues
  • Improving documentation

Using the Data

All data in ToxBase is freely available for research, educational, and public health purposes. When using ToxBase data in publications or other work, please acknowledge the source and consider contributing back any additional data you generate.

The database can be used for various purposes, including:

  • Conducting meta-analyses across multiple toxicology studies
  • Identifying dose-response relationships for specific toxins
  • Comparing outcomes across different animal models
  • Supporting regulatory risk assessments
  • Educational purposes in toxicology training
